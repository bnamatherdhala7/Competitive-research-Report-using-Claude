Under the hood

The tech stack

Each component is deterministic. Only the AI reasoning layer is probabilistic.

🔍

Brave Search API

Web research — pricing pages, product comparisons, reviews. $0 cost for up to 2,000 queries/month.

r/

Reddit JSON API

Community sentiment scraping across r/AdobeCreative, r/Firefly, r/Midjourney. No auth required.

▶

YouTube Data API v3

Video reviews and comparison sentiment. Top 8 results per product query.

🤖

Claude Haiku

Synthesis and structured analysis. 73% cheaper than Sonnet. Target cost <$0.012 per report.

⚡

Trigger.dev

Cloud orchestration. Runs the full pipeline — scrape → analyze → output — in under 60 seconds.

▲

Vercel

Static hosting for all output HTML. Auto-deploys on every GitHub push. Zero config.
